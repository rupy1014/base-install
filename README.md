# Base Install

개발 환경 설치 스크립트

## 설치 방법

### Windows

PowerShell을 **관리자 권한**으로 실행한 후 아래 명령어를 입력하세요:

```powershell
irm https://git.clauders.ai/jobdori/base-install/raw/main/window-install.ps1 | iex
```

### Mac

터미널을 열고 아래 명령어를 입력하세요:

```bash
curl -fsSL https://git.clauders.ai/jobdori/base-install/raw/main/mac-install.sh | bash
```
