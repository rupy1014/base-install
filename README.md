# Base Install

Claude Code 원클릭 설치 스크립트 - OS에 맞는 명령어를 복사해서 붙여넣으세요.

## Mac / Linux

터미널을 열고:

```bash
curl -fsSL https://gitea.max5.ai/ehowlsla/base-install/raw/branch/main/mac-install.sh | bash
```

## Windows

PowerShell을 **관리자 권한**으로 열고:

```powershell
irm https://gitea.max5.ai/ehowlsla/base-install/raw/branch/main/window-install.ps1 | iex
```
